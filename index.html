Google Teachable Machine is a no-code tool from Google that lets you train a computer vision model by showing it example images for each class. After training, the model looks at each live webcam frame, compares visual patterns like color and shape with what it learned, and returns confidence scores to predict whether the wine bottle appears to be White or Red Wine, or Rosé.
